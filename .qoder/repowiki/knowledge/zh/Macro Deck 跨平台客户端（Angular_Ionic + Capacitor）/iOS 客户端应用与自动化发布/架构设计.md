- **混合架构核心**：采用 Capacitor 作为跨平台桥接层，`AppDelegate.swift` 通过 `ApplicationDelegateProxy` 将原生生命周期事件委托给 Web 视图容器。
- **依赖管理**：使用 CocoaPods 管理原生依赖，包括基础 Capacitor 运行时及屏幕方向、条码扫描、设备信息等社区插件。
- **自动化流水线**：通过 Fastlane 定义标准化工作流，利用 `Matchfile` 从 Git 仓库同步证书与配置文件，实现 `build`（构建 IPA）与 `release`（上传 TestFlight）的自动化闭环。
- **配置集中化**：应用标识符、权限描述（如相机使用）及界面方向限制集中在 `Info.plist` 与 `Podfile` 中管理。
