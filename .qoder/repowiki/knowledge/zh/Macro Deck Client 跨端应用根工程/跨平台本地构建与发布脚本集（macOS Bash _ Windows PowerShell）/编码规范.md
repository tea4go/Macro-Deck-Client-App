- 所有 Shell 脚本顶部启用 set -euo pipefail，PowerShell 脚本设置 $ErrorActionPreference = 'Stop'，确保错误即时中断。
- 外部命令调用前必须经 require_command / Require-Command 校验存在性，缺失时输出中文安装提示并退出。
- 构建流程严格遵循「Web 资源构建 → Capacitor 同步 → 原生平台打包」三段式顺序，由 _common.* 中的 prepare_web_build / ionic_build / cap_sync 统一编排。
- Windows 脚本通过 dot-sourcing 引入 _common.ps1，复用 Write-Ok/Warn/Fail/Banner 日志、Confirm-* 交互确认、Invoke-NativeStream 原生命令包装等公共能力。
- 敏感配置一律放在 scripts/local/ 下的 *.example 模板文件，禁止提交至版本控制；CI 通过仓库 Secrets 注入同名环境变量覆盖本地值。
