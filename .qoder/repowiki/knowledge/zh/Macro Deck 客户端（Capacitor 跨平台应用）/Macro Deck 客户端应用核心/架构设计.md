- **根模块引导与 PWA 支持**：`AppModule` 统一集成 `IonicModule`、`IonicStorageModule` 及 `ServiceWorkerModule`，通过懒加载机制（`HomePageModule`, `DeckPageModule`）组织业务页面，并在 Web 环境下启用离线缓存。
- **分层服务架构**：将底层通信（`WebSocketService`）、协议解析（`Protocol2Service`）、硬件适配（`ScreenOrientation`, `WakeLock`）及业务逻辑（`ConnectionService`, `MacroDeckService`）封装为独立 Injectable 服务，实现关注点分离。
- **事件驱动的状态同步**：`MacroDeckService` 作为核心状态容器，通过 `EventEmitter` 广播配置更新与用户交互事件；`Protocol2Service` 监听交互流并转换为协议消息，形成“UI -> Service -> Protocol -> WebSocket”的单向数据流闭环。
- **平台差异化启动**：`AppComponent` 根据环境标识动态选择根页面（Web/Home），并处理深度链接（Deep Linking）以支持快速连接二维码扫描，同时在原生平台跳过 SSL 验证配置。
