- 首次环境引导需按序执行 `install_1_base_tools_*` → `install_2_ruby_*` → `install_3_fastlane_*` → `install_4_android_sdk_*`（macOS 使用 brew，Windows 使用 winget）。
- Android 签名：复制 `scripts/local/android-signing.env.example` 为 `android-signing.env`（macOS）或 `android-signing.ps1`（Windows），至少填写 `KEYSTORE_FILE_PASSWORD`；Windows 构建时 `versionCode` 自动递增，iOS 需额外设置 `MATCH_PASSWORD` 及 App Store Connect API 密钥环境变量。
